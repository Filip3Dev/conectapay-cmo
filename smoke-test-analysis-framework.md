### Detailed Decision Breakdown

#### ✅ GO (Build Product) if:

**Volume**:
- [ ] 500-1.000+ leads generated
- [ ] 50-100+ qualified leads
- [ ] 20-50+ paying customers (or strong intent)

**Quality**:
- [ ] 50%+ of leads score 7-10
- [ ] 60%+ WTP > R$ 300/mo
- [ ] 80%+ recognize the problem
- [ ] 30%+ ready to buy now

**Efficiency**:
- [ ] CAC < R$ 100
- [ ] LTV:CAC > 3x
- [ ] Payback < 3 months

**Validation**:
- [ ] Clear value proposition resonates
- [ ] Winning channel(s) identified
- [ ] Pricing validated
- [ ] ICP clearly defined

**Action**: Proceed to MVP development, hire 2-3 beta customers

---

#### ⚠️ PIVOT (Refine Strategy) if:

**Volume OK, Quality Issues**:
- [ ] 200-500 leads generated
- [ ] 20-50 qualified leads
- [ ] CAC R$ 100-150 (acceptable but not great)

**Quality Problems**:
- [ ] 30-50% score 7-10 (below target)
- [ ] 40-60% WTP > R$ 300/mo (some validation)
- [ ] Mixed feedback on value prop

**Efficiency Marginal**:
- [ ] CAC R$ 100-150
- [ ] LTV:CAC 2-3x (marginal)
- [ ] Payback 3-4 months

**Validation Weak**:
- [ ] Some channels work, others don't
- [ ] Messaging unclear or inconsistent
- [ ] Pricing not fully validated

**Action**: Run second smoke test with revised messaging/positioning

---

#### ❌ STOP (Not Validated) if:

**Volume Low**:
- [ ] < 200 leads generated
- [ ] < 20 qualified leads
- [ ] < 5 paying customers (or weak intent)

**Quality Poor**:
- [ ] < 30% score 7-10
- [ ] < 40% WTP > R$ 300/mo
- [ ] < 60% recognize the problem
- [ ] < 20% ready to buy

**Efficiency Bad**:
- [ ] CAC > R$ 150
- [ ] LTV:CAC < 2x
- [ ] Payback > 4 months

**Validation Failed**:
- [ ] No clear value prop
- [ ] No winning channels
- [ ] Pricing rejected
- [ ] ICP unclear

**Action**: Return to interviews, test new niche, or pivot problem hypothesis

**Short-term (Month 1-2)**:
5. [ ] Build MVP (WhatsApp automation + Pix)

7. [ ] Iterate based on feedback
8. [ ] Prepare for broader launch

**Medium-term (Month 3-6)**:
9. [ ] Launch to 50-100 customers

11. [ ] Hire customer support
12. [ ] Optimize unit economics

---

### If PIVOT (Refine Strategy)

**Immediate (Week 1-2)**:
1. [ ] Analyze why quality/efficiency was marginal
2. [ ] Identify top 3 hypotheses to test

4. [ ] Test new creative variations

5. [ ] Launch second smoke test
6. [ ] Focus on winning channel(s) only
