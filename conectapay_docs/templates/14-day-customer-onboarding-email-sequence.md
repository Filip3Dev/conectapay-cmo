# 14-Day Customer Onboarding Email Sequence

**Purpose**: Guide new customers from signup to activation and first success
**Audience**: New ConectaPay customers (all segments)
**Frequency**: 6 emails over 14 days
**Channel**: Email + WhatsApp (optional parallel sequence)

---

## Onboarding Journey Overview

```
Day 0 → Welcome + Setup
Day 1 → Quick Win Configuration
Day 3 → Check-in
Day 7 → Milestone Celebration
Day 14 → Completion + Next Steps
```

**Expected Outcomes**:
- 90% complete setup by Day 3
- 80% send first message by Day 7
- 70% recover first abandoned sale by Day 14

## Engagement Triggers

### When to Send Extra Emails

**Trigger 1: No Activity in 48 Hours**
Send: "Oi [Name]! Vi que você ainda não configurou. Tudo bem?"

**Trigger 2: First Message Sent**
Send: "Parabéns! Primeira mensagem enviada 🎉"

**Trigger 3: First Sale Recovered**
Send: "🎉 PRIMEIRA VENDA RECUPERADA! Você já recuperou R$ [X]"

**Trigger 4: Low Response Rate (<20%)**
Send: "Sua taxa de resposta está abaixo da média. Quer ajuda para otimizar?"

**Trigger 5: High Response Rate (>50%)**
Send: "Uau! Sua taxa de resposta está excelente! Quer duplicar suas campanhas?"

## Technical Setup

### Email Service Provider
- **Recommended**: HubSpot (free tier for first 2,000 emails/month)
- **Alternative**: Mailchimp, SendGrid, Amazon SES

### Automation Triggers
1. **Signup** → Send Email 0 immediately
2. **24 hours later** → Send Email 1
3. **72 hours later** → Send Email 2
4. **7 days later** → Send Email 3
5. **14 days later** → Send Email 4

### HubSpot Workflow Setup
1. Create workflow: "Customer Onboarding"
2. Trigger: Contact lifecycle stage = "Customer"
3. Add delays and email sends
4. Set goal: Created first campaign
5. Branch based on goal completion
