## Decision Matrix: What to Work on Next

When deciding what to work on, ask these questions in order:

1. **Does this require customer input?** If yes → DO ICP interviews first
2. **Does this validate assumptions?** If yes → DO smoke test first
3. **Does this unblock other work?** If yes → DO infrastructure first
4. **Can this be delegated?** If yes → Delegate to contractor
5. **Is this nice-to-have?** If yes → STOP, do critical work first

**Examples**:
- ❌ "Should I create a referral program?" → NO (no customers yet)
- ✅ "Should I send ICP interview DMs?" → YES (unblocks all other work)
- ❌ "Should I write more content?" → NO (enough content, 0% published)
- ✅ "Should I build the landing page?" → YES (unblocks smoke test)
