## How to Use These Documents

**For Advocacy Strategy Planning**:
1. Start with **CON-219** (Program Design) for overall strategy
2. Review **CON-198** (Launch Plan) for execution approach
3. Reference **CON-169** (Referral Launch) for referral-specific planning

**For Advocacy Program Launch**:
1. Use **CON-219** to understand program vision and components
2. Follow **CON-198** week-by-week launch checklist
3. Launch referral program using **CON-169** phased approach

**For CS Organizational Assessment**:
1. Use **CON-222** (Maturity Model) to assess current state across 8 domains
2. Identify capability gaps and prioritize improvements
3. Follow implementation roadmap (Phase 1-5) for sequential development
4. Track progress monthly using maturity assessment template

**For CS Operations Planning**:
1. Assess current maturity using **CON-222** scoring framework
2. Reference domain-specific strategies (Onboarding, Health Scoring, etc.)
3. Use operational templates from **CON-210** for implementation
4. Align CS investments with maturity level progression (don't skip levels)
