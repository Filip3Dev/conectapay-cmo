## Instagram Outreach Strategy

### Target Profiles
- **Clínicas**: Search `clínica [cidade]` + `consulta` + `agendamento`
- **Imobiliárias**: Search `imobiliária [cidade]` + `aluguel` + `venda`

### DM Script (Portuguese)

```
👋 Olá! Tudo bem?

Vi que você é gestor(a) na @NOME_EMPRESA. Estou pesquisando os desafios que clínicas/imobiliárias enfrentam hoje com:

📌 Clínicas: Pacientes que não comparecem às consultas (no-show)
📌 Imobiliárias: Leads que não convertem em vendas

Você teria 20min para uma conversa rápida essa semana? **Não é venda** - é pesquisa para entender melhor o problema e validar uma solução.

Posso te enviar o link do calendário? 📅
```

### Follow-up Day 3 (if no response)
```
👋Oi novamente! Sei que está corrido...

Só reforçando: são apenas 20min, sem pitch de vendas. Quero entender sua dor real com no-show/conversão de leads.

Vale um café virtual ☕ em troca do seu tempo?

Calendário: [LINK]
```

### Follow-up Day 7
```
Última tentativa! 😅

Seu feedback seria valioso para shapear a solução. 20min, zoom ou whatsapp.

Se não der agora, tudo bem - vou fechar essa rodada de pesquisas.

Abraço!
```

## LinkedIn Outreach Strategy

### Target Filters (Sales Navigator)
- **Title**: "Sócio", "Diretor", "Gerente", "Corretor"
- **Industry**: "Real Estate" ou "Health Care"
- **Company Size**: 11-200 employees
- **Location**: São Paulo, Rio de Janeiro, Belo Horizonte, Curitiba, Salvador

```
Olá [NOME],

Vi que você lidera a [EMPRESA] aqui em [CIDADE].
```
