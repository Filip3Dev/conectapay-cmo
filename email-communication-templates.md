   - Menos ruído = melhor gravação

3. Seja honesto

## Email Setup Guide

### Option 1: Calendly Integrations
1. Go to Calendly > Integrations
2. Connect email provider (Gmail/Outlook)
3. Set up automated notifications
4. Customize templates in Calendly

### Option 2: Mailchimp / ActiveCampaign
1. Create email templates
2. Set up automation triggers
3. Connect Calendly webhook
4. Personalize with merge fields

### Option 3: Manual (For Initial Phase)
1. Save templates as drafts
2. Use find/replace for variables
3. Copy/paste for each email
4. Track in spreadsheet

## Best Practices

### Timing
- Confirmation: Immediately (automated)
- Reminder: 24h before (automated)
- Thank you: Within 1h (manual)
- Beta invite: When ready (manual)
- Referral: 7 days later (manual)
- Re-engagement: 30-45 days (manual)

### Personalization
- Always use their name
- Reference something specific from interview
- Use their exact words when possible
- Mention their company name

### Tone
- Professional but casual
- Portuguese Brazilian
- Short and to the point
- Respect their time
